# 🌸 ZERA BOT 2026 - V2.0

Zera is a high-speed WhatsApp bot with the latest 2026 features. It is designed to run 24/7 on platforms like Railway or VPS.

## 🚀 Key Features
- ⚡ **0-6 Seconds Response**
- 🔑 **Pairing Code Login** (No QR needed)
- 🖼️ **Sticker Maker** (Image/Video to Sticker)
- 📥 **All-in-One Downloader** (Insta Reels, YouTube, Status)
- 🎙️ **Text to Speech** (Malayalam/English)
- 📞 **Truecaller Information** & Live Location
- ☁️ **Cloud Storage** for media files

## 🛠️ Setup & Deployment

1. **Fork** this repository.
2. Open `index.js` and update your phone number in the `phoneNumber` section.
3. Deploy the repository to **Railway.app**.
4. Check Railway **Logs** for your **Pairing Code**.
5. Open WhatsApp > Linked Devices > Link with Phone Number and enter the code.

## 🤖 Commands
| Command | Description |
| --- | --- |
| `.alive` | Check if Zera is online |
| `.menu` | Display all features |
| `.sticker` | Convert image to sticker |
| `.insta` | Download Instagram Reels |
| `.tts` | Text to Voice Note |
| `.ping` | Check bot speed |
